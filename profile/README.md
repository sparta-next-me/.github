## 🚀 프로젝트 소개

**Next Me는 AI 기반 미래 자산 설계 서비스입니다.**

사용자가 금융 계좌를 연동하고 미래 자산 목표를 설정하면,
AI가 소비, 저축, 자산 흐름을 분석하여
현재 재무 상태를 파악할 수 있는 분석 결과를 제공합니다.<br>
본 서비스는 목표를 자동으로 관리하는 방식이 아니라,
설정한 목표 대비 자산 흐름을 비교, 점검할 수 있는
기준과 분석 결과를 제공하는 데에 목적을 둡니다.<br>
또한 적금 상품 추천,
결제 후 이용 가능한 전문가 상담 예약,
RAG 기반 AI 챗봇 기능을 통해
자산 관리에 필요한 정보를 하나의 서비스에서 제공합니다.

## 🎯 주요 기능

### 계좌 연동 및 자산 데이터 수집

- 사용자 금융 계좌 연동
- 입출금 내역, 잔액, 금융 상품 데이터 수집
- 스케줄러 기반 데이터 갱신
- 데이터 정형화

### 목표 설정 및 AI 분석

- 사용자 목표 저장
- 거래 내역 데이터 분석
- 목표 달성을 향한 분석 결과 제공
- 자산 상태를 참고한 상품 추천

### AI 챗봇

- 사용자 데이터 기반 질의 처리
- 대화형 질의응답 인터페이스 제공

### 전문가 상담 결제 및 1:1 채팅
- 상담 상품 조회
- 결제 후 상담 예약 처리
- 예약 완료 시 1:1 채팅방 생성

### 선착순 프로모션 및 대기열 처리

- 선착순 포인트 이벤트 처리
- 대기열 기반 요청 처리
- 요청 제어를 통한 매크로 방지
